# Chatty

### Overview

A simple SPA chatroom.

### Dependencies

* React
* Webpack
* [babel-loader](https://github.com/babel/babel-loader)
* [webpack-dev-server](https://github.com/webpack/webpack-dev-server)
* randomcolor
* uuid
